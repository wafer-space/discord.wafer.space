# The submission deadline submitted for Run #2 is 2026-07-15 11:59 a.m.!

If you are planning to be on this shuttle, please make sure that all of these things are done **before** the deadline (Wednesday, July 15, 2026 11:59:00 a.m.):
 * GDS has been uploaded to https://platform.wafer.space and has successfully **passed the current precheck** version on the platform and has been "submitted for manufacturing".
 * Added your CrowdSupply Order number to your project.
 * Set the chip on board setting correctly.

The chip on board setting changes the flags given to the precheck command and verifies that your design has the correct pad layout to be used with the wafer.space chip on board setup.

The queue for prechecks running on the platform can get long as the deadline gets closer and the precheck can be quite slow, so **do not leave it to the last minute**.

Even if your GDS is not yet ready **you should still create your project and set the CrowdSupply Order number as soon as possible.** I have already set the value for people who I could match manually -- but there 12 orders who do not have a project associated with them.

If you do not have manufacturable GDS submitted before the deadline, you will be getting blank silicon die. **There is no option for refund or being deferred to the next shuttle.**
